# React User Dashboard

A fully responsive User Dashboard application built using React.  
This project demonstrates API integration, global state management, routing, and persistent storage using localStorage.



##  Features

- Fetch users from API
- Display users in cards
- Search users (Name, Email, Phone, City)
- Create new users
- User Details page using React Route
- Persistent data using localStorage (data remains after refresh)
- Clean and responsive UI with Tailwind CSS



##  Tech Stack

- React (Functional Components + Hooks)
- Context API (Global State Management)
- Axios (API calls)
- React Router DOM
- Tailwind CSS
- localStorage



##  Installation

1. Clone the repository

git clone <your-github-link>

2. Install dependencies

3.cd User-dashboard

4.npm install

5. Run project

npm run dev

 http://localhost:5173
